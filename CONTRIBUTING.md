# 贡献指南

感谢你愿意参与本项目的建设！请遵循以下流程：

## 开发流程

1. 在 Issues 中找到一个未分配的任务，或创建新 Issue 描述你要做的工作。
2. 在本地基于 `main` 分支创建功能分支：
   
   git checkout -b feature/你的功能名称

3. 完成代码开发，确保功能正常运行。

4. 提交代码并推送到远程仓库：

   
   git add .
   git commit -m "✨ 描述你的修改"
   git push origin feature/你的功能名称

5. 在 GitHub 上创建 Pull Request，目标分支为 main。

6. 等待至少 1 名其他成员 Review 通过后即可合并
